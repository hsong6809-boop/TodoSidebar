Todo Sidebar 项目说明
====================

项目位置：C:\Users\ADMIN\miclaw\project\TodoSidebar

项目介绍
--------
这是一个基于 WPF (.NET 8.0) 的 Todo 任务管理应用，具有两种模式：
1. 完整模式（FullWindow）- 可添加、删除、完成任务
2. 侧边栏模式（MainWindow）- 只能完成任务，自动隐藏在屏幕左侧

核心功能：
- 每日任务管理
- 截止日期任务管理
- 每周任务管理（可折叠）
- 历史完成任务记录
- 里程碑管理
- 任务优先级（高/中/低）
- 任务计数器

技术栈：
- .NET 8.0 WPF
- SQLite 本地存储
- CommunityToolkit.Mvvm (MVVM 框架)
- Win32 API (毛玻璃效果)
